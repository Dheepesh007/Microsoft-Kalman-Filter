# 🪟 Microsoft Kalman Filter (State Estimation)

## 🧠 Overview

This project demonstrates a simplified Kalman Filter concept used in robotics and AI. It estimates the system state by combining a predicted value and a measured value.

This concept is widely used in Microsoft technologies for tracking, navigation, and AI systems.

---

## 🎯 Objective

To compute the estimated state using:

Estimate = (Prediction + Measurement) / 2

---

## 📊 Dataset Used

* Kalman Filter Demo Dataset
  🔗 https://github.com/rlabbe/Kalman-and-Bayesian-Filters-in-Python

---

## ⚙️ Implementation

* Language: Python
* Platform: Google Colab / VS Code
* Concept: State Estimation (Kalman Filter Simplification)

---

## 🧮 Example

* Prediction = 20
* Measurement = 24

**Output:**
Estimated State = 22

---

## 🏭 Industry Application

Used in:

* Robotics
* Navigation systems
* AI tracking
